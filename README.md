# morse-code-translator
This is a morse code translator that will translate an input file to/from morse code and write it to an output file.
It supports the following characters:
- A - Z
- 0 - 9
- '!', '?', ',', '.', ';', '(', ')', '_', and ':'

Example usage to encode:
```bash
morse_code_encoder input.txt output.txt
```

Example usage to decode:
```bash
morse_code_decoder input.txt output.txt
```
